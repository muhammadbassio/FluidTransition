FluidTransition (swift)
=================

A customized Presentation stack for UIKit supporting custom animated transitioning as well as interactive transitioning

The main target of developing this library is to overcome https://openradar.appspot.com/29840481 


## Installation

##### CocoaPods

[CocoaPods](http://cocoapods.org) is a dependency manager for Cocoa projects. You can install it with the following command:

```bash
$ gem install cocoapods
```
To integrate Alamofire into your Xcode project using CocoaPods, specify it in your `Podfile`:

```ruby
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '11.0'
use_frameworks!

target '<Your Target Name>' do
    pod 'FluidTransition'
end
```

Then, run the following command:

```bash
$ pod install
```
##### Manual

Drag the folder "source" to your Xcode project.
